# Sistema de Registro de Vehículos 

## Descripción

Esta aplicación permite registrar vehículos dentro de un garaje mediante una interfaz gráfica desarrollada con Tkinter en Python.
El sistema permite ingresar información de vehículos y visualizar los registros dentro de una tabla.

## Funcionalidades

* Ingresar información de un vehículo.
* Registrar el vehículo en el sistema.
* Visualizar los vehículos registrados en una tabla.
* Limpiar los campos del formulario.

## Datos del vehículo

Cada vehículo registrado contiene la siguiente información:

* Placa
* Marca
* Propietario

### Descripción de carpetas

* modelos
Contiene la clase que representa los datos del vehículo.

* servicios
Contiene la lógica del programa para gestionar los vehículos.

* ui
Contiene la interfaz gráfica desarrollada con Tkinter.

* main.py
Archivo principal que inicia la aplicación.
